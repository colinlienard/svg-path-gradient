Instructions

  1. Drop an SVG file with a single path into the drop zone
  2. Add or modify gradient color stops with position percentages (0-100)
  3. Adjust the stroke width as needed
  4. Set the number of gradient segments (more segments = smoother gradient)
  5. Click "Export as PNG" to download the result

Note: This tool creates gradients that truly follow the path's direction.
